## gopeed

支持多协议（HTTP、BitTorrent）下载的客户端，提供命令行、RESTful API、WebSocket、Go 类库方式来使用。

## 安装

```sh
go install github.com/monkeyWie/gopeed
```

## 示例

可以在本仓库的[\_examples](_examples)目录下查看
